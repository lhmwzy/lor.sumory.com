Get started with Lor Framework

Quick start to install lor and build up a lor app

# Clone the repository
$ git clone https://github.com/sumory/lor

# Go into the repository
$ cd lor

# before install lor, you should have installed OpenResty and luajit
# now install lor in /opt/lua/lor
# the `lor` cli will be in /usr/local/bin/
$ sh install.sh /opt/lua/lor #choose your path as you want

# use lor to build a project scaffold
$ cd /tmp/ && lor new lor_demo && cd /tmp/lor_demo
$ lor start
# then open http://localhost:8888/ in some browser
